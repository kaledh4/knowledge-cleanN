## Automatic Deployments:

Every time you push to the main branch, GitHub Actions will:
1. Build the Next.js app
2. Export static files to /out folder
3. Deploy to GitHub Pages automatically

## Troubleshooting:

If deployment fails:
- Check that secrets are set correctly in repository settings
- Verify the workflow completed successfully in Actions tab
- Check that Pages is enabled with "GitHub Actions" as the source
- Look at workflow logs for specific errors
